        <summary><span class="training-no">1</span> 접지시스템의 구성요소</summary>
        <div class="training-body">
          <div class="training-grid">
            <article><h4>보호도체(PE)</h4><p>감전 방지 등 안전을 위해 노출도전부와 주 접지단자를 연결하는 도체입니다. 전기기계기구의 철대와 금속 외함은 전원의 자동차단 방식으로 보호할 때 PE에 확실히 접속해야 합니다.</p></article>
            <article><h4>보호등전위본딩 도체</h4><p>건축물 내부로 들어오는 금속 배관, 철골·철근 등 서로 다른 도전부 사이의 전위차를 줄이기 위해 연결하는 도체입니다.</p></article>
            <article><h4>접지도체</h4><p>계통·설비·기기의 한 점과 접지극 사이를 연결하는 도전성 경로입니다. 접지극의 모든 접지도체 연결지점에는 영구적으로 고정되는 ‘안전전기 연결’ 표시가 필요합니다.</p></article>
            <article><h4>보조 보호등전위본딩 도체</h4><p>자동차단시간을 만족하기 어렵거나 전기설비와 직접 관련은 없지만 감전에 기여할 수 있는 금속체를 추가로 본딩할 때 사용합니다.</p></article>
          </div>
          <div class="tip-box"><strong>현장 점검 포인트</strong><p>PE, 접지도체, 본딩도체를 모두 같은 ‘접지선’으로만 보지 말고 목적과 연결 대상이 맞는지 확인합니다. 노출도전부와 계통외도전부의 구분이 핵심입니다.</p></div>
        </div>
      </details>

      <details class="training-section">
        <summary><span class="training-no">2</span> TT계통 접지 기준과 누전차단기</summary>
        <div class="training-body">
          <p>TT계통은 전원의 한 점을 직접 접지하고, 설비의 노출도전부를 전원측 접지극과 전기적으로 독립된 접지극에 접속하는 방식입니다. 누전차단기를 사용하는 경우 아래 조건을 만족해야 합니다.</p>
          <div class="formula-box">R<sub>A</sub> × I<sub>Δn</sub> ≤ 50V</div>
          <p class="small">R<sub>A</sub>: 보호도체와 접지극 저항의 합(Ω) · I<sub>Δn</sub>: 누전차단기 정격감도전류(A)</p>
          <div class="table-wrap"><table class="report-table"><thead><tr><th>정격감도전류</th><th>30mA</th><th>50mA</th><th>100mA</th><th>300mA</th><th>500mA</th><th>1,000mA</th></tr></thead><tbody><tr><th>접지저항 한계값</th><td>1,667Ω</td><td>1,000Ω</td><td>500Ω</td><td>167Ω</td><td>100Ω</td><td>50Ω</td></tr></tbody></table></div>
          <div class="warning-box"><strong>주의</strong><p>여러 누전차단기가 본딩된 하나의 접지망을 공유하면 설치된 정격감도전류 가운데 가장 큰 값으로 계산한 가장 엄격한 접지저항을 적용합니다. 누전차단기 보호를 받지 못하는 계량기함·배전반 등은 100Ω 이하를 권장합니다.</p></div>
          <div class="example-box"><strong>예시</strong><p>30mA와 1,000mA 누전차단기가 동일 접지망에 연결되어 있다면 1,000mA 기준인 50Ω 이하를 적용합니다. 개별 접지라면 각 회로의 감도전류별 기준을 적용할 수 있습니다.</p></div>
        </div>
      </details>

      <details class="training-section">
        <summary><span class="training-no">3</span> TN-C-S 계통과 추가접지</summary>
        <div class="training-body">
          <p>TN-C-S 계통은 인입구까지 PEN도체를 사용하고, 인입점 이후 N과 PE를 분리합니다. PEN도체가 단선되거나 개방되면 설비 외함에 위험전압이 나타날 수 있으므로 인입구 추가접지극의 시설 여부가 중요합니다.</p>
          <ul class="training-list"><li>N과 PE가 분리된 뒤 하류에서 다시 접속되어 있지 않은지 확인</li><li>인입구 PEN 분리점의 추가접지극 시설 여부 확인</li><li>추가접지가 없으면 사용전점검에서 부적합 판정 가능</li><li>추가접지 기준값은 명확한 적부판정값이 없으나 100Ω 이하 권장</li></ul>
          <div class="tip-box"><strong>실무 TIP</strong><p>분전반 하류에서 N-PE가 재결합되면 정상 부하전류가 PE로 흐르고 누전차단기 오동작 또는 외함 전위 상승의 원인이 됩니다.</p></div>
        </div>
      </details>

            <tr><td>보조 보호등전위본딩</td><td>Cu 2.5㎟(기계적 보호) / 4㎟(미보호)</td><td>연결되는 더 작은 보호도체 단면적 이상 등 검토</td></tr>
          </tbody></table></div>
          <div class="memory-box"><strong>보호도체 간편 선정</strong><p>선도체 S ≤ 16㎟ → PE는 S와 동일 · 16㎟ &lt; S ≤ 35㎟ → PE 16㎟ · S &gt; 35㎟ → PE는 S/2</p></div>
          <div class="warning-box"><strong>사용전점검 지적 사례</strong><p>접지도체와 PE 최소 굵기 혼동, PEN도체 굵기 부족, 본딩 대상 금속배관 누락, 기계적 보호가 없는 2.5㎟ PE 사용 등이 대표적입니다.</p></div>

      function showUpdateNotice(registration) {
        if (document.getElementById('pwaUpdateNotice')) return;
        var box = document.createElement('div');
        box.id = 'pwaUpdateNotice';
        box.className = 'pwa-update-notice';
        box.innerHTML = '<span>새 버전이 있습니다.</span><button type="button">업데이트</button>';

          if (registration && registration.waiting) {
            registration.waiting.postMessage({ type: 'SKIP_WAITING' });
          } else {
            window.location.reload();
          }
        });
      }

      if ('serviceWorker' in navigator) {
        var refreshing = false;
        navigator.serviceWorker.addEventListener('controllerchange', function () {
          if (refreshing) return;
          refreshing = true;
          window.location.reload();
        });

        window.addEventListener('load', function () {
          navigator.serviceWorker.register('./service-worker.js').then(function (registration) {
            if (registration.waiting && navigator.serviceWorker.controller) {
              showUpdateNotice(registration);
            }
            registration.addEventListener('updatefound', function () {
              var newWorker = registration.installing;
              if (!newWorker) return;
              newWorker.addEventListener('statechange', function () {
                if (newWorker.state === 'installed' && navigator.serviceWorker.controller) {
                  showUpdateNotice(registration);
                }
              });
            });
          }).catch(function (err) {
            console.warn('Service worker registration failed:', err);
          });
        });
      }
