부하 기준 자재 선정

부하용량(kW)을 기준으로 전류를 계산한 뒤 차단기·케이블·터미널·전선관을 추천합니다. HP는 모터 출력 기준 보조 입력입니다.

전선관 부속 선정

전선관 종류와 규격을 입력하면 커넥터·인서트·새들·홀커터 참고 규격을 표시합니다.

케이블 자재 선정

케이블 SQ를 기준으로 현장 구매용 터미널 규격을 1개만 추천합니다.

단자대·터미널 선정

단자대 정격과 극수, 사용할 케이블 SQ를 선택하면 대표 터미널 규격을 1개만 추천합니다.

전력절감 계산

입력은 최소화하고, 절감 방식은 운전시간 개선과 전력량 절감 두 가지로만 계산합니다. 기후환경요금·연료비조정요금은 자동 적용해 결과에만 표시합니다.

운전시간 개선

가동 시간은 여러 구간 입력이 가능합니다. 예: 08:00-11:00, 13:00-15:00. 입력한 시간은 검침월 계절 기준으로 경부하·중간부하·최대부하에 자동 배분되어 결과에만 표시됩니다.

설비 설치현황

정확성을 위해 설비명, 부하용량, 대수는 명판 또는 현장 확인값을 직접 입력합니다. 여러 설비를 추가하면 총 부하가 자동 합산됩니다.

등록된 설비가 없습니다.

전력량 절감

가동 시간은 여러 구간 입력이 가능합니다. 예: 08:00-11:00, 13:00-15:00. 경부하·중간부하·최대부하 운전시간은 자동 계산되어 결과에만 표시됩니다.

가동 시간 클릭 선택

시작 시간을 누른 뒤 종료 시간을 누르면 선택한 입력란에 자동 추가됩니다. 30분 단위 선택이 가능합니다.
